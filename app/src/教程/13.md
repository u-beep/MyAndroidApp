## 一、知识点1：单选框 RadioGroup + RadioButton
### 作用
一组选项**只能选一个**，比如性别：男 / 女，二选一。
- `RadioGroup`：分组容器
- `RadioButton`：单个选项

### 基础写法
```xml
<RadioGroup
    android:id="@+id/rg_sex"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:orientation="horizontal">

    <RadioButton
        android:id="@+id/rb_man"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="男"/>

    <RadioButton
        android:id="@+id/rb_woman"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="女"/>
</RadioGroup>
```
- `orientation="horizontal"`：横向排列；vertical：上下竖排

### Kotlin获取选中值
```kotlin
val rgSex = findViewById<RadioGroup>(R.id.rg_sex)
//获取选中id
val selectId = rgSex.checkedRadioButtonId
var sex = "男"
if(selectId == R.id.rb_woman){
    sex = "女"
}
```

## 三、知识点3：ScrollView 滚动布局
### 作用
页面控件太多，超出手机屏幕高度时，**手指上下滑动查看全部内容**
> 规则：**ScrollView内部只能有一个直接子布局**（常用LinearLayout / ConstraintLayout包裹所有控件）

基础结构模板：
```xml
<ScrollView
    android:layout_width="match_parent"
    android:layout_height="match_parent">

    <!-- 唯一一层子布局，所有内容放这里 -->
    <LinearLayout
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:orientation="vertical"
        android:padding="15dp">

        所有输入框、单选、复选、按钮全部写在这

    </LinearLayout>
</ScrollView>
```

## 五、修改 RegisterActivity.kt：把性别、爱好一起存入SP
原来只存账号-密码，现在：**账号=密码|性别|爱好** 拼接字符串存储
```kotlin
import android.content.SharedPreferences
import android.os.Bundle
import android.widget.*
import androidx.appcompat.app.AppCompatActivity

class RegisterActivity : AppCompatActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_register)

        val etAccount = findViewById<EditText>(R.id.et_account)
        val etPwd = findViewById<EditText>(R.id.et_pwd)
        val etPwd2 = findViewById<EditText>(R.id.et_pwd2)
        val btnReg = findViewById<Button>(R.id.btn_register)

        //单选
        val rgSex = findViewById<RadioGroup>(R.id.rg_sex)
        //多选
        val cbBall = findViewById<CheckBox>(R.id.cb_ball)
        val cbBook = findViewById<CheckBox>(R.id.cb_book)
        val cbGame = findViewById<CheckBox>(R.id.cb_game)

        btnReg.setOnClickListener {
            val acc = etAccount.text.toString().trim()
            val pwd = etPwd.text.toString().trim()
            val pwd2 = etPwd2.text.toString().trim()

            //判空
            if(acc.isEmpty()||pwd.isEmpty()||pwd2.isEmpty()){
                Toast.makeText(this,"信息不能为空",Toast.LENGTH_SHORT).show()
                return@setOnClickListener
            }
            if(pwd!=pwd2){
                Toast.makeText(this,"两次密码不一致",Toast.LENGTH_SHORT).show()
                return@setOnClickListener
            }

            //获取性别
            var sex = "男"
            if(rgSex.checkedRadioButtonId == R.id.rb_woman){
                sex = "女"
            }

            //获取爱好
            var hobby = ""
            if(cbBall.isChecked) hobby += "篮球 "
            if(cbBook.isChecked) hobby += "看书 "
            if(cbGame.isChecked) hobby += "游戏 "
            if(hobby.isEmpty()) hobby = "无"

            //保存：密码|性别|爱好
            val sp = getSharedPreferences("USER_LIST",MODE_PRIVATE)
            if(sp.contains(acc)){
                Toast.makeText(this,"账号已存在",Toast.LENGTH_SHORT).show()
                return@setOnClickListener
            }
            val saveStr = "$pwd|$sex|$hobby"
            sp.edit().putString(acc,saveStr).apply()

            Toast.makeText(this,"注册成功",Toast.LENGTH_SHORT).show()
            finish()
        }
    }
}
```
