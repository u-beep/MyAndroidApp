## 🐱 小宠物功能一览

| 互动方式 | 效果 |
|---|---|
| **点击宠物** | 随机说话 + 切换开心表情（😺😸😻😽） |
| **拖动宠物** | 可以拖到屏幕任意位置 |
| **拖动结束** | 有概率说"别推我嘛~" |
| **自动眨眼** | 每3~6秒随机切换表情 |
| **进入主页** | 自动打招呼"喵~你好！" |

## 📋 改动文件

| 文件 | 说明 |
|---|---|
| `PetView.kt` | **新建** - 自定义宠物控件（emoji表情 + 气泡对话 + 拖拽 + 自动动画） |
| `activity_main.xml` | 外层包了 `FrameLayout`，添加宠物浮层容器 |
| `MainActivity.kt` | 初始化宠物、放在左下角、onDestroy释放资源 |

## 🔑 核心知识点

- **自定义 View**：继承 `View`，重写 `onDraw()` 用 Canvas+Paint 画图
- **触摸事件**：`onTouchEvent` 区分点击和拖动（通过移动距离阈值）
- **定时动画**：`Handler.postDelayed` 实现眨眼和气泡消失
- **层叠布局**：`FrameLayout` 让宠物浮在内容上方
- **防内存泄漏**：`onDestroy` 中调用 `release()` 清除 Handler 回调
