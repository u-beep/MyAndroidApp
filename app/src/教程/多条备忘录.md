| `MemoEditActivity.kt` | 备忘录编辑/新建页面（保存到SP） |
| `activity_memo_edit.xml` | 编辑页面布局（输入框 + 保存按钮） |

### 📝 修改文件（4个）

| 文件 | 改动 |

| `AndroidManifest.xml` | 注册 `MemoEditActivity` |

### 🔑 核心设计

1. **按账号独立存储**：每个账号有独立的SP文件 `MEMO_zhangsan`、`MEMO_lisi`，互不干扰

2. **存储格式**：
   ```
   SP文件：MEMO_账号
   key   = 备忘录ID（时间戳，如 1692000000000）
   value = "内容|时间"（如 "今天天气不错|2026-06-06 14:30"）
   ```

3. **新建 vs 编辑**：
    - 新建：不传 `MEMO_ID` → 自动用 `System.currentTimeMillis()` 生成新ID
    - 编辑：传入 `MEMO_ID` + `MEMO_CONTENT` → 覆盖原ID的数据

4. **防误删**：删除时弹出 `AlertDialog` 确认弹窗

### 🚀 使用流程

1. 登录进入主页 → 点击「备忘录」
2. 看到**备忘录列表**（最新的在最前面）
3. 点击「+ 新建备忘录」→ 输入内容 → 保存 → 回到列表
4. 点击某条备忘录 → 进入编辑模式 → 修改内容 → 保存
5. 点击某条的「删除」→ 弹出确认弹窗 → 确认删除
6. 不同账号登录，看到各自的备忘录，互不影响
