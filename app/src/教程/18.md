## 一、知识点1：资源拆分管理（规范开发必备）
### 1.1 colors.xml（res/values/colors.xml）统一存放所有颜色
原来XML里到处写`#2196F3`，全部抽离，后续改色只改一处
```xml
<?xml version="1.0" encoding="utf-8"?>
<resources>
    <color name="colorPrimary">#2196F3</color>
    <color name="colorGreen">#4CAF50</color>
    <color name="colorOrange">#FF9800</color>
    <color name="colorGray">#9E9E9E</color>
    <color name="white">#FFFFFF</color>
    <color name="bg_page">#F5F5F5</color>
    <color name="text_black">#333333</color>
</resources>
```
使用方式：`@color/colorPrimary`

### 1.2 dimens.xml（res/values/dimens.xml）统一尺寸dp
```xml
<?xml version="1.0" encoding="utf-8"?>
<resources>
    <dimen name="dp_10">10dp</dimen>
    <dimen name="dp_15">15dp</dimen>
    <dimen name="dp_20">20dp</dimen>
    <dimen name="dp_25">25dp</dimen>
    <dimen name="text_15">15sp</dimen>
    <dimen name="text_18">18sp</dimen>
    <dimen name="text_25">25sp</dimen>
</resources>
```
使用：`@dimen/dp_20`

### 1.3 替换项目原有XML
示例按钮：
```xml
<Button
    style="@style/MyButton"
    android:text="登录"
    android:backgroundTint="@color/colorPrimary"/>
```
输入框边距：`android:layout_marginTop="@dimen/dp_10"`

## 二、知识点2：全局Theme主题，统一APP页面风格
打开`res/values/themes/themes.xml`，修改应用主题：
```xml
<style name="Theme.MyApplication" parent="Theme.MaterialComponents.DayNight.NoActionBar">
    <!-- 标题栏底色 -->
    <item name="colorPrimary">@color/colorPrimary</item>
    <!-- 页面默认背景 -->
    <item name="android:windowBackground">@color/bg_page</item>
    <!-- 控件默认着色 -->
    <item name="colorAccent">@color/colorOrange</item>
</style>
```
效果：整个APP所有页面统一背景色，系统控件默认色调统一。

## 三、知识点3：自定义圆角条目背景（drawable新建item_bg.xml）
给RecyclerView列表条目做圆角卡片样式
```xml
<?xml version="1.0" encoding="utf-8"?>
<shape xmlns:android="http://schemas.android.com/apk/res/android"
    android:shape="rectangle">
    <solid android:color="@color/white"/>
    <corners android:radius="12dp"/>
</shape>
```
打开`item_user.xml`，根布局添加背景+内边距：
```xml
<LinearLayout 
    android:background="@drawable/item_bg"
    android:padding="@dimen/dp_15"
    android:layout_margin="@dimen/dp_10">
```
列表条目变成白色圆角卡片，界面整洁美观。

## 四、知识点4：自定义Dialog弹窗（重点）
原生弹窗样式丑，自定义布局弹窗，用于**删除用户确认弹窗、修改密码提示**
### 步骤1：新建弹窗布局 dialog_delete.xml
```xml
<?xml version="1.0" encoding="utf-8"?>
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:orientation="vertical"
    android:padding="@dimen/dp_20"
    android:background="@drawable/item_bg">

    <TextView
        android:id="@+id/tv_msg"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="确定要删除该用户吗？"
        android:textSize="@dimen/text_18"
        android:textColor="@color/text_black"/>

    <LinearLayout
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:orientation="horizontal"
        android:layout_marginTop="@dimen/dp_20">

        <Button
            android:id="@+id/btn_cancel"
            android:layout_width="0dp"
            android:layout_height="wrap_content"
            android:layout_weight="1"
            android:text="取消"
            android:backgroundTint="@color/colorGray"/>
        <Button
            android:id="@+id/btn_confirm"
            android:layout_width="0dp"
            android:layout_height="wrap_content"
            android:layout_weight="1"
            android:text="确定删除"
            android:backgroundTint="@color/colorOrange"/>
    </LinearLayout>
</LinearLayout>
```

### 步骤2：RecyclerView条目长按弹出删除弹窗，实现删除功能
#### 1.UserAdapter增加长按回调
```kotlin
var longClick:((User)->Unit)?=null
//onBindViewHolder内部
holder.itemView.setOnLongClickListener {
    longClick?.invoke(user)
    true
}
```
#### 2.MainActivity绑定长按事件
```kotlin
userAdapter.longClick = { selectUser ->
    //构建自定义弹窗
    val dialog = android.app.Dialog(this)
    dialog.setContentView(R.layout.dialog_delete)
    dialog.window?.setLayout(550, android.view.ViewGroup.LayoutParams.WRAP_CONTENT)

    val btnCancel = dialog.findViewById<Button>(R.id.btn_cancel)
    val btnConfirm = dialog.findViewById<Button>(R.id.btn_confirm)

    //取消关闭弹窗
    btnCancel.setOnClickListener { dialog.dismiss() }
    //确定删除数据库数据
    btnConfirm.setOnClickListener {
        val dao = UserDao(this)
        dao.deleteUser(selectUser.account)
        //刷新列表
        userList.clear()
        userList.addAll(dao.getAllUser())
        userAdapter.notifyDataSetChanged()
        dialog.dismiss()
        Toast.makeText(this,"删除成功",Toast.LENGTH_SHORT).show()
    }
    dialog.show()
}
```
