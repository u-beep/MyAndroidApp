## 一、第一步：图片资源存放规则（必记）
1. 在项目目录：`res → drawable` 文件夹，把png/jpg图片复制粘贴进去
> 命名规则：**小写英文、数字、下划线，不能中文/空格**，例：`logo.png、head.png`
2. 代码引用格式：`@drawable/图片名`，不用写后缀

### 三种加载图片方式
1. 本地静态图片（drawable里）：`android:src="@drawable/logo"`
2. 项目矢量图标：`@drawable/ic_xxx`（AS自带Image Asset生成）
3. 网络图片（后面课程学，今天不学）

## 二、ImageView核心常用属性详解
|属性|作用|
|----|----|
|android:layout_width|控件宽度，match_parent铺满/ wrap_content自适应/自定义dp|
|android:layout_height|控件高度|
|android:src|设置图片资源（原图尺寸载入）|
|android:scaleType|图片缩放方式，控制图片过大过小变形问题|

### scaleType6种常用模式（重点，背4个）
1. **fitXY**：拉伸铺满控件，**图片大概率变形**，不推荐做头像
2. **centerCrop**：等比放大图片，裁掉超出部分填满控件（头像首选！）
3. **fitCenter**：等比缩放，完整显示图片，留白（产品配图常用）
4. **centerInside**：图片小于控件就居中，大于控件等比缩小完整显示

## 三、实操1：改造登录页，顶部加LOGO头像
打开`activity_login.xml`（约束布局），在标题`tv_title`上方添加头像控件
```xml
<!-- 顶部用户头像 -->
<ImageView
    android:id="@+id/iv_head"
    android:layout_width="100dp"
    android:layout_height="100dp"
    android:src="@drawable/head"
    android:scaleType="centerCrop"
    app:layout_constraintTop_toTopOf="parent"
    app:layout_constraintStart_toStartOf="parent"
    app:layout_constraintEnd_toEndOf="parent"
    android:layout_marginTop="30dp"/>
```
> 绑定约束：上下左右相对父布局居中，距离顶部30dp

标题控件修改约束，跑到头像下方：
```xml
<TextView
    android:id="@+id/tv_title"
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:text="用户登录"
    android:textSize="30sp"
    app:layout_constraintTop_toBottomOf="@id/iv_head"
    app:layout_constraintStart_toStartOf="parent"
    app:layout_constraintEnd_toEndOf="parent"
    android:layout_marginTop="20dp"/>
```

## 四、实操2：制作圆形头像（drawable新建xml）
1. 右键drawable→New→Drawable Resource File，命名`circle_head.xml`
```xml
<?xml version="1.0" encoding="utf-8"?>
<shape xmlns:android="http://schemas.android.com/apk/res/android"
    android:shape="oval">
    <!--oval=椭圆，宽高一致就是正圆-->
    <solid android:color="#ffffff"/>
</shape>
```
2. ImageView添加背景，配合centerCrop实现圆形头像
```xml
<ImageView
    android:id="@+id/iv_head"
    android:layout_width="100dp"
    android:layout_height="100dp"
    android:src="@drawable/head"
    android:background="@drawable/circle_head"
    android:scaleType="centerCrop"
    android:clipToOutline="true"
    app:layout_constraintTop_toTopOf="parent"
    app:layout_constraintStart_toStartOf="parent"
    app:layout_constraintEnd_toEndOf="parent"
    android:layout_marginTop="30dp"/>
```
> `clipToOutline="true"`：按照背景形状裁剪图片，实现圆形

## 五、实操3：圆角矩形图片（圆角配图）
drawable新建`radius_img.xml`
```xml
<?xml version="1.0" encoding="utf-8"?>
<shape xmlns:android="http://schemas.android.com/apk/res/android"
    android:shape="rectangle">
    <solid android:color="#fff"/>
    <!--radius统一圆角大小，dp越大圆角越大-->
    <corners android:radius="15dp"/>
</shape>
```
需要圆角的ImageView引用`android:background="@drawable/radius_img"`

## 六、代码动态修改图片（Kotlin代码修改，拓展）
在LoginActivity.kt的onCreate中，可动态替换图片
```kotlin
//找到图片控件
val ivHead = findViewById<ImageView>(R.id.iv_head)
//更换图片
ivHead.setImageResource(R.drawable.logo)
```
