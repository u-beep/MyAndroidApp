# 📅 Android第15天详细教程｜RecyclerView列表控件（零基础专用）
## 今日学习总目标
RecyclerView是安卓**最核心列表控件**，通讯录、聊天列表、商品列表全靠它；今天实现：**用户注册信息列表展示**，把所有注册账号、性别、爱好在主页列表全部显示，掌握列表布局、数据绑定、条目点击。

> 前置：之前所有注册信息存在`USER_LIST`的SharedPreferences，今天读取数据装载进列表。

## 一、前置配置（必须先加依赖）
打开项目app模块下 `build.gradle(Module级别)`，在dependencies里面添加依赖：
```gradle
dependencies {
    //RecyclerView依赖
    implementation "androidx.recyclerview:recyclerview:1.3.2"
}
```
添加后点击右上角 **Sync Now** 同步依赖。

## 二、RecyclerView三要素（必背）
1. **RecyclerView**：列表容器（放页面上，承载所有条目）
2. **Item布局xml**：单条数据长什么样（一条用户信息布局）
3. **Adapter适配器**：桥梁，**数据源 ↔ 列表条目**，负责赋值、创建条目

### 步骤2：创建数据实体类 User.kt（包目录右键新建Kotlin Class）

### 步骤3：创建适配器 UserAdapter.kt（核心适配器）
```kotlin
import android.view.LayoutInflater
import android.view.ViewGroup
import androidx.recyclerview.widget.RecyclerView
import com.example.myapplication.databinding.ItemUserBinding

class UserAdapter(var list:MutableList<User>):RecyclerView.Adapter<UserAdapter.UserVH>() {

    //条目ViewHolder，缓存控件
    inner class UserVH(val binding:ItemUserBinding):RecyclerView.ViewHolder(binding.root)

    //创建条目布局
    override fun onCreateViewHolder(parent: ViewGroup, viewType: Int): UserVH {
        val inflate = LayoutInflater.from(parent.context)
        val binding = ItemUserBinding.inflate(inflate,parent,false)
        return UserVH(binding)
    }

    //绑定数据到控件
    override fun onBindViewHolder(holder: UserVH, position: Int) {
        val user = list[position]
        holder.binding.tvAcc.text = "账号：${user.account}"
        holder.binding.tvSex.text = "性别：${user.sex}"
        holder.binding.tvHobby.text = "爱好：${user.hobby}"
    }

    //返回列表数据总数
    override fun getItemCount(): Int = list.size
}
```
> 若没开启viewBinding，改用findViewById写法，需要我替换纯find版适配器随时说。

### 步骤5：MainActivity.kt读取SP数据、绑定列表
```kotlin
import android.content.SharedPreferences
import androidx.recyclerview.widget.LinearLayoutManager
import com.example.myapplication.User
import com.example.myapplication.UserAdapter

//定义集合存放所有用户
var userList = mutableListOf<User>()
lateinit var userAdapter:UserAdapter

override fun onCreate(savedInstanceState: Bundle?) {
    super.onCreate(savedInstanceState)
    setContentView(R.layout.activity_main)

    //1.绑定RecyclerView
    val rvList = findViewById<RecyclerView>(R.id.rv_user_list)
    //设置布局管理器：竖直列表
    rvList.layoutManager = LinearLayoutManager(this)
    //初始化适配器
    userAdapter = UserAdapter(userList)
    rvList.adapter = userAdapter

    //2.从SP读取全部注册用户
    val sp:SharedPreferences = getSharedPreferences("USER_LIST",MODE_PRIVATE)
    val allKeys = sp.all.keys //获取所有注册账号(key)

    //循环拆分数据存入集合
    userList.clear()
    for(key in allKeys){
        val infoStr = sp.getString(key,"") ?: continue
        val arr = infoStr.split("|")
        val pwd = arr[0]
        val sex = arr[1]
        val hobby = arr[2]
        //封装对象加入集合
        userList.add(User(key,pwd,sex,hobby))
    }
    //刷新列表
    userAdapter.notifyDataSetChanged()
}
```

## 四、运行效果
1. 提前注册若干账号（带性别、爱好）
2. 登录进入主页，下方自动**竖向列表展示全部注册用户信息**
3. 数据来自本地SharedPreferences，重启APP数据仍存在

```kotlin
}
```

## 🎯 今日必背三句话
1. **RecyclerView=容器，Item=单行布局，Adapter=数据桥梁**
2. LinearLayoutManager → 竖直/横向列表（默认竖向）
3. `notifyDataSetChanged()`：刷新列表数据
