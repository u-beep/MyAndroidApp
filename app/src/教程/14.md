# 一、先学：ProgressBar 加载进度条
## 作用
登录、注册时**显示正在加载**，不让用户干等。

## 基础写法（直接放 XML）
```xml
<!-- 圆形加载条，默认转圈 -->
<ProgressBar
    android:id="@+id/progressBar"
    android:layout_width="40dp"
    android:layout_height="40dp"/>
```

## 常用两种样式
1. **圆形转圈**（默认）
2. **水平进度条**（下载用）

我们今天只用**圆形加载**。

# 二、实战 1：登录页加加载动画
打开 `activity_login.xml`，在**登录按钮下面**加：

```xml
<ProgressBar
    android:id="@+id/loading"
    android:layout_width="40dp"
    android:layout_height="40dp"
    android:visibility="gone"/>  <!-- 默认隐藏 -->
```

> `gone = 隐藏`
> `visible = 显示`

---

# 三、实战 2：点击登录 → 显示加载
打开 `LoginActivity.kt`

## 1. 顶部定义控件
```kotlin
lateinit var loading: ProgressBar
```

## 2. onCreate 里绑定
```kotlin
loading = findViewById(R.id.loading)
loading.visibility = View.GONE // 一开始隐藏
```

## 3. 点击登录时：显示加载
```kotlin
btn_login.setOnClickListener {
    // 第一步：显示加载动画
    loading.visibility = View.VISIBLE
    btn_login.isEnabled = false // 禁止重复点击

    // 你的登录逻辑...（账号密码判断）

    // 登录成功/失败后
    loading.visibility = View.GONE // 隐藏加载
    btn_login.isEnabled = true
}
```

## 效果
**一点登录 → 按钮变灰 + 转圈加载 → 成功/失败后停止**

# 四、实战 3：Style 样式抽取（今天最重要！）
## 什么是 Style？
把**所有控件共用的样式**写在一个地方，**改一次，全部生效**！

比如：
- 所有按钮颜色一样
- 所有输入框圆角一样
- 所有文字大小一样

不用每个控件复制粘贴！

---

# 五、第一步：创建 Style
打开 `res/values/styles.xml`

添加**按钮样式**：

```xml
<style name="MyButton">
    <item name="android:layout_width">match_parent</item>
    <item name="android:layout_height">wrap_content</item>
    <item name="android:backgroundTint">#2196F3</item>
    <item name="android:textColor">#ffffff</item>
    <item name="android:padding">12dp</item>
    <item name="android:layout_marginTop">10dp</item>
</style>
```

添加**输入框样式**：

```xml
<style name="MyEditText">
    <item name="android:layout_width">match_parent</item>
    <item name="android:layout_height">wrap_content</item>
    <item name="android:padding">10dp</item>
    <item name="android:layout_marginTop">10dp</item>
</style>
```

# 六、第二步：XML 直接使用（超级简洁！）
## 原来的按钮（又臭又长）
```xml
<Button
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:text="登录"
    android:backgroundTint="#2196F3"
    android:textColor="#fff"
    android:padding="12dp"/>
```

## 现在的按钮（只用一行）
```xml
<Button
    style="@style/MyButton"  <!-- 直接用样式！ -->
    android:text="登录"/>
```

## 输入框也一样
```xml
<EditText
    style="@style/MyEditText"
    android:hint="请输入账号"/>
```

# 七、实战 4：给登录页一键美化（完整版）
把你的 `activity_login.xml` 改成这样，**干净又整齐**：

```xml
<?xml version="1.0" encoding="utf-8"?>
<ScrollView
    xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent">

    <LinearLayout
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:orientation="vertical"
        android:padding="20dp">

        <!-- 头像 -->
        <ImageView
            android:layout_width="100dp"
            android:layout_height="100dp"
            android:src="@drawable/logo"
            android:layout_gravity="center"/>

        <!-- 标题 -->
        <TextView
            android:layout_width="wrap_content"
            android:layout_height="wrap_content"
            android:text="用户登录"
            android:textSize="25sp"
            android:layout_gravity="center"
            android:layout_marginTop="20dp"/>

        <!-- 账号 -->
        <EditText
            style="@style/MyEditText"
            android:id="@+id/et_account"
            android:hint="请输入账号"/>

        <!-- 密码 -->
        <EditText
            style="@style/MyEditText"
            android:id="@+id/et_pwd"
            android:inputType="textPassword"
            android:hint="请输入密码"/>

        <!-- 记住密码 -->
        <CheckBox
            android:id="@+id/cb_remember"
            android:layout_width="wrap_content"
            android:layout_height="wrap_content"
            android:text="记住密码"/>

        <!-- 登录按钮 -->
        <Button
            style="@style/MyButton"
            android:id="@+id/btn_login"
            android:text="登录"/>

        <!-- 注册按钮 -->
        <Button
            style="@style/MyButton"
            android:id="@+id/btn_to_register"
            android:text="去注册"
            android:backgroundTint="#607D8B"/>

        <!-- 加载动画 -->
        <ProgressBar
            android:id="@+id/loading"
            android:layout_width="40dp"
            android:layout_height="40dp"
            android:visibility="gone"
            android:layout_gravity="center"/>

    </LinearLayout>
</ScrollView>
```
